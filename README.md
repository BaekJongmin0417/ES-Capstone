### ES-Capstone
(2020-2) ES-Entertainment SW Media Project Capstone Design <br>
이금형, 백종민, 윤강민 <br><br><br>
#### 프로젝트 개요
프로젝트 명: 플래시 블루<br>
플랑크톤을 표현한 미디어아트에 코로나19와 환경 변화에 관한 메시지를 담았다.<br>
자연환경의 변화를 프로세싱과 키넥트, LED를 사용해서 시각화하고 관객 참여로 확장해 나간다.<br>
플랑크톤을 자연 생태계, 밤하늘, 불꽃놀이, 은하수 등을 연상되게 표현하고 COVID-19와 우울감(blue)이 합쳐진 신조어인 ‘코로나 블루’라는 단어처럼 전체적인 프로젝트의 색감을 파란 계열로 표현한다.<br>
해변을 방문한 관광객에 의해 파괴되던 생태계가 COVID-19 이후 관광객이 줄며 복구된 상황을 본 따, 관객 인식 전에 자유롭게 유영하던 플랑크톤이 관객 인식 시 파괴되는 것처럼 터지는 모습을 보여준다. <br><br><br>
#### 개발 환경
processing: 플랑크톤의 모양과 움직임, 변화를 구현<br>
Kinect v2: 관객 참여 형식으로 구현<br>
arduino, sensor, LED: 관객에 반응하여 파도가 치는 모습을 구현<br><br><br>
